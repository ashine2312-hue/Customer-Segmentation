E-commerce enables online buying and selling for both consumers and businesses, but the lack of face-to-face interaction makes it harder to understand customer preferences. This project addresses that gap by analyzing historical sales data to understand consumer behavior. It uses RFM (Recency, Frequency, Monetary) based K-means clustering for customer segmentation and applies machine learning models for purchase prediction. Among the models tested, a combination of Logistic Regression and XGBoost achieved the best performance in improving sales insights.
